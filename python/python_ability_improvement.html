    # 说明

    非常推荐！！！

    重构可以帮助你代码更简洁，写出的代码思路更清晰

    粗浅的学过之后，对自己写代码帮助挺大的，至少整体会有一种宏观的想法，最简单的：

    函数的功能要尽可能单一，不能将一个简单函数写成既有赚钱的功能，又有顾家的功能，应该将两个功能拆成两个不同的函数

    博客搭建系列之一---windows安装yarn

    zlbingo 12/13/2021 blog

    本文采用vuepress来搭建博客，官网教程推荐使用yarn来进行安装部署

    # 1. 安装nodejs

    通过查看官方yarn的windows安装文档 (opens new window)

    It is recommended to install Yarn through the npm package manager (opens new window), which comes bundled with Node.js (opens new window) when you install it on your system.

    官方说需要先安装Node.js，那我们就先安装Node.js

    # 1.1 下载软件

    通过Node.js 中文网 (nodejs.cn) (opens new window)下载最新版。

    # 1.2 安装

    • 直接点下一步，在选择安装路径的时候选择自己规划的路径，建议在下面这个图的时候，选中Automatically install ...，这样的话可以帮助我们自动下载一些必备工具。

    • 安装完成之后会弹出PowerShell，帮你安装一些工具例如Chocolatey (opens new window)（这个工具可以帮助我们安装yarn）

    # 1.3 验证安装

    打开PowerShell或者cmd，输入node --version,可以看到如下图所示，即代表安装成功

    # 2 安装yarn

    官方给出三种windows下安装yarn的方式

    • 直接下载yarn的msi安装包

      前提安装好Node.js

    • 通过Chocolatey安装

      前提安装好Node.js

    • 通过Scoop安装

    由于前面我们已经安装好了Chocolatey，因此这里我们以Chocolatey为例示范安装

    # 2.1 通过Chocolatey安装

    直接通过该命令，然后根据提示输入A即可

    choco install yarn
输入A
    1
    2

    # 2.2 验证安装

    打开PowerShell或者cmd，输入yarn --version,可以看到如下图所示，即代表安装成功
